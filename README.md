## 🚀 Getting Started

### Installation

Clone the repository:
``` bash
git clone https://github.com/YassineIdiri/ExpressJS_Backend_OnlineGame.git
cd ExpressJS_Backend_OnlineGame
```

Install dependencies:
```bash
npm install
```

 Start the development server:
```bash
npm start
```
